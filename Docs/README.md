## Nyx-OS


# What is Nyx OS
Nyx OS is a browser-based personal PC/AI assistant built using HTML, CSS and JavaScript

It can remember informationn, store it locally, can find it for you, respond to commands, has a mood based background colour, and provides a chat style interface

# Why I built it
I wanted to learn Web development also had to make a Ai assistant of mine and this seemed a good idea to me.

Also i didnt know much of JavaScript and CSS so with the help of this project i was able t learn those skills as well

And also make a cool project

# Current Features
- Memory system
- Memory search
- Memory clear
- Name storage
- Mood based theme and background
- Typing animation (Nyx's side)
- Help command  
- Status bar

# How it works
- Nyx uses JavaScript to process user commands
- Data entered by user is saved locally if told by the user
- When you put a specific command the mood may chang like for "help" command Nyx changes mood to "helper"
- Currently Nyx is running on browser and doesnt need a backend server
