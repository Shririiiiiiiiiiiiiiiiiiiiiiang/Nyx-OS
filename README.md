## Nyx-OS


# What is Nyx OS
Nyx OS is a browser-based persnolpc/ai assistantbuilt using HTML,CSS,JavaScript

It can remember info, store it locally, can find it for you, respond to commands, has a mood based background colour, and provides a chat style interface

# Why I built it
I wanted to learn Web development also had to make a Ai assistant of mine and this seemed a good idea to me.

Also i didnt know much of JavaScript and CSS so with the help of this project i was able t learn those skills as well

And also make a cool project

# Current Features
- Memory system
- Memory search
- Memory clear
- Name storage
- Mood based theme and background
- Typing animation (Nyx's side)
- Help command
- Status bar


# Tech Stack
- HTML
- CSS
- JavaScript
- LocalStorage
